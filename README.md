# Образец универсального чат бота для образовательных учреждений
Данный проект преднозначен для общения с технической поддержкой образовательного учреждения для которой он будет развёрнут и
размещения информации (новостей, оповещений и т.п.)
>Проект разработан на базе Node.js, СУБД postgreSQL и telegram API
Все настройки и токены расположениы в файле ```config.js```
```
export const sys_admin_phone = ''
export const sys_admin_name = '/ваше имя/'
export const sys_admin_token = '/sha256 id профиля админа/'

export const PORT = /int порт сервера бота/;
export const TOKEN = "/токен бота из @BotFather/"
```
## Требования к системуе
### Для работы бота потребуется:
* Node.js v20.16.0
* npm v9.1.1
* postgreSql v16.3.2
